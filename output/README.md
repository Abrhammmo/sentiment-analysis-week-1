## Stock Sentiment Analysis Project
** Overview of this output folder 

This project analyzes the relationship between news sentiment and stock returns for major tech companies (AAPL, AMZN, GOOG, META, MSFT, NVDA). It calculates daily sentiment scores from cleaned analyst headlines, merges them with stock price data, and explores correlations using statistical and visual analyses.


Data Sources

- Stock Price Data: Historical stock prices from Yahoo Finance.

- News Data: Analyst ratings headlines (cleaned and preprocessed).

## Tasks
    - Task 2: Stock Indicators

    - Computes stock indicators for each ticker.

    - Saves results as CSV and PNG plots.

## Task 3: Sentiment Analysis & Correlation

    - Cleans headlines and calculates daily sentiment scores using VADER.

    - Merges sentiment with daily stock returns.

    - Computes and visualizes correlations (scatter plots and rolling correlations).

    - Saves results as CSV, PNG plots, and OLS summary.
